News Marquee Tool for LMS: Template Showcase

Welcome to this demo of a news marquee tool, which can be embedded into an LMS using iframes. This demo is currently configured for a Music Business curriculum, but the layout adapts automatically to any field by editing a single configuration file.

🍴 Fork This Template on GitHub

How this works: A background automated GitHub Action updates this repository every 4 hours, scraping live RSS data models directly from a feed of your choice. This example scrapes Billboard, Music Business Worldwide, and Digital Music News. This data populates a static news.json payload, completely bypassing typical browser CORS security restrictions and delivering real-time, zero-maintenance topical touchpoints inside Canvas iFrames.
